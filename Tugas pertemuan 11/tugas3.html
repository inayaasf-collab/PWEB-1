FAQ Interaktif (jQuery)

Apa itu HTML?
HTML adalah bahasa markup untuk membuat struktur halaman web.
Apa itu CSS?
CSS digunakan untuk memberi gaya pada halaman web.
Apa itu JavaScript?
JavaScript adalah bahasa pemrograman yang membuat web menjadi interaktif.
Apa itu jQuery?
jQuery adalah library JavaScript yang mempermudah manipulasi DOM.
Apa itu DOM?
DOM adalah representasi struktur dokumen yang dapat dimanipulasi dengan skrip.
